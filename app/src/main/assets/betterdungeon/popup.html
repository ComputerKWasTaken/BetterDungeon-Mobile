Command Mode

Send narrative commands to the AI

Adds a Command button to the input menu. Text is formatted as ## Your Command: for direct AI instructions.

Great for time skips, scene changes, and guiding story direction.

Standard## Your Command:
Subtle[## Your Command:]
OOC((OOC ... | User: Your Question?))

Use ↑↓ arrow keys while in Command mode to switch between Standard, Subtle, and OOC sub-modes.

Standard — direct story instructions. Subtle — indirect guidance via brackets. OOC — ask the AI a direct question.

Try Mode

RNG-based action outcomes

RNG determines if actions succeed or fail. Formats as: You try to [action], you [result].

Outcomes: critically fail, fail, succeed, critically succeed.

Critical Chance 5%

Scripture

Live visual UI for script-driven displays

Scripture lets adventure scripts render interface elements outside the story text, including stat bars, counters, panels, status readouts, and custom HTML widgets.

Useful for RPG HUDs, relationship meters, timers, inventory panels, and scenario-specific dashboards.

WebFetch

Consent-gated web requests and simple search

WebFetch gives scripts conservative web access for live context: safe GET requests, basic search, origin allow/deny decisions, rate limits, and blocked local/private targets.

Useful for current facts, public reference pages, lightweight lookups, and web-aware scripted tools.

Origin Permissions

Clock

Reliable time, time zones, and formatting

Clock gives scripts the user's current local time, IANA time-zone conversion, ISO timestamps, offsets, and deterministic formatted dates.

Useful for daily events, timed encounters, schedules, cooldowns, holidays, and world clocks.

SDK

BetterDungeon version and configuration context

SDK gives scripts curated BetterDungeon metadata that complements heartbeat without duplicating Ultrascripts discovery, including version info and safe configuration context.

Useful when a script needs to react to how the player has BetterDungeon configured while leaving heartbeat as the one module/ops truth.

World & Device Context

4/4

Geolocation

Permission-based real-world location context

Geolocation lets scripts request the user's browser-approved location, returning coordinates, accuracy, and permission state only after the normal browser prompt allows it.

Useful for local mysteries, travel scenes, real-world scavenger hooks, and location-aware setting details.

Weather

Current conditions and forecast lookup

Weather gives scripts a focused weather path without general web scraping, using place names or coordinates to return current conditions and short forecasts.

Useful for atmospheric scenes, survival mechanics, travel planning, seasonal events, and grounded local flavor.

Network

Connection status and quality hints

Network reports online status and browser connection hints like effective type, round-trip time, downlink estimate, and data-saver preference when available.

Useful for scripts that can choose lighter requests, avoid live lookups, or explain degraded behavior cleanly.

System

Coarse device, browser, display, and power hints

System exposes broad environment details such as device class, platform family, locale, screen size, touch support, browser family, and battery state when supported.

Useful for adapting script UI, reducing work on mobile or low-power devices, and tailoring experiences by device context.
